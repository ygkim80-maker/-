# 의약품 연관사업 인사이트 모니터링 에이전트

의약품(제약·바이오) 산업과 그 **연관사업**(CDMO/CMO, 원료의약품(API), 헬스케어 플랫폼·디지털 헬스, 건강기능식품, 화장품, 동물의약품 등)에서 나오는 시장 인사이트와 사업 아이템을 매일 자동으로 리서치하여 `reports/` 폴더에 리포트로 쌓아두는 모니터링 에이전트입니다.

## 무엇을 하는가

매일 1회, 다음 카테고리를 중심으로 최신 뉴스·산업 동향·기업 사례를 웹 검색으로 수집하고 요약합니다.

1. **CDMO/위탁생산** — 국내외 CDMO 시장 규모, 신규 수주, 설비 투자, 세포유전자치료제(CGT) CDMO 등
2. **원료의약품(API)** — 수출입 동향, 고부가가치 원료, 공급망 이슈
3. **디지털 헬스케어 플랫폼** — 제약사의 디지털 헬스 진출, DTx, 웨어러블 연동, AI 헬스케어
4. **연관 다각화 사업** — 건강기능식품, 화장품, 동물의약품 등 제약사의 신사업 진출 사례
5. **정책·규제 변수** — 약가, 인허가, 미국/중국 등 주요국 정책 변화가 사업 기회에 미치는 영향

각 실행 결과는 `reports/YYYY-MM-DD.md` 파일로 커밋되어 히스토리로 남습니다. GitHub 저장소에서 날짜별로 과거 리포트를 열람하고, 커밋 로그로 변경 이력을 추적할 수 있습니다.

## 동작 방식

- 이 에이전트는 Claude Code의 스케줄링 기능(cron)을 통해 **매일 1회** 실행되도록 설정되어 있습니다.
- 실행될 때마다 `prompts/daily-research-prompt.md`에 정의된 리서치 지침에 따라 웹 검색을 수행하고, 결과를 정리해 `reports/YYYY-MM-DD.md`로 커밋·푸시합니다.
- 스케줄은 이 작업을 생성한 세션(컨테이너)에 종속되며 **최대 7일 후 자동 만료**됩니다. 세션이 종료되거나 컨테이너가 회수되면 예약 실행도 함께 중단됩니다. 장기적으로 계속 자동 실행되길 원한다면 Claude Code on the web의 **Trigger(예약 실행)** 기능으로 별도 등록하는 것을 권장합니다. (https://code.claude.com/docs)

## 폴더 구조

```
README.md                        이 문서
prompts/daily-research-prompt.md 매일 실행되는 리서치 지침(프롬프트) 원본
reports/YYYY-MM-DD.md            날짜별 리서치 리포트
```
